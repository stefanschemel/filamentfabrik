Bestellverlauf

Eine Bestellung bei der Filamentfabrik - So funktioniert es!

  • Schritt 1: Projekt kostenfrei anfragen

    Teilen Sie der Filamentffabrik ihr Anliegen per Mail mit. Die Filamentfabrik wird ihnen innerhalb weniger Stunden entweder ein Angebot übermitteln oder Sie fachmännisch Beraten.

  • Schritt 2: Auftrag erteilen

    Nachdem Sie das professionelle Angebot der Filamentfabrik erhalten haben, ist es notwendig, dass Sie uns ein Auftrag erteilen. Dies können Sie kinderleicht per Mail machen. Nach Eingang iherer Bestellung erhalten sie eine Auftragsbestätigung.

  • Schritt 3: Produktion

    Je nach Auftrag beginnt die Filamentfabrik nach Auftragseingang entweder mit der Bauteilauslegung oder der Bauteilproduktion. Der Kunde erhält in sinnvollen Abständen Updates über den Status der Bestellung.

  • Schritt 4: Qualitätskontrolle & Versand

    Die Qualität der Produkte spielt in der Filamentfabrik eine große Rolle. Nur Produkte, deren Qualität einwandfrei ist, werden an den Kunden versand.

  • Schritt 5: Bezahlung

    Sie können die Produkte entweder per Überweisung oder per Paypal bezahlen nach erhalt der Ware bezahlen. Sollte das Auftragsvolumen 75€ übersteigen, so wird bereits bei Auftragseingang eine Anzahlung von 45% der Bestellung fällig.
